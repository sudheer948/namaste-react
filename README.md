# 📦 Parcel

Parcel is a **zero-config bundler** used to bundle and optimize your React application.

## ✨ Features of Parcel

- Dev Build
- Local Server
- HMR = Hot Module Replacement
- File Watching Algorithm (written in C++)
- Caching → Faster Builds
- Image Optimization
- Minification
- Bundling
- Compression
- Consistent Hashing
- Code Splitting
- Differential Bundling → Supports older browsers
- Diagnostic
- Error Handling
- HTTPS Support
- Tree Shaking → Removes unused code
- Different Dev and Production Bundles
